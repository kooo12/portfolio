Instructions:

  1. Click the download buttons to generate your portfolio icons
  2. Replace the following files in your web folder:
    • favicon.png - Use the favicon download
    • icons/Icon-192.png - Use the 192x192 download
    • icons/Icon-512.png - Use the 512x512 download
  3. Copy the downloaded files to replace the existing Flutter icons
  4. Rebuild your Flutter web app: flutter build web
